# Bujamart - Your Online Marketplace

**Bujamart** is an online shopping platform designed to serve customers in Burundi. It offers a variety of products ranging from fashion to home appliances, and more. The platform provides an easy-to-use shopping experience with an option for **Pay on Delivery** and the ability to pay via **PayPal**.

## Features

- **Browse Products**: Explore a variety of categories like men's and women's clothes, footwear, home appliances, and more.
- **Shopping Cart**: Add products to your cart and proceed to checkout.
- **Order Management**: Admin dashboard for managing orders.
- **Multi-language Support**: Available in **English** and **French**.

## Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript**
- **Firebase Authentication** for user management
- **Firebase Firestore** for storing cart data and orders

## How to Run

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/bujamart.git
    ```

2. Open the `index.html` file in your browser to start using the platform.
